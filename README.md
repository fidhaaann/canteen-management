# Canteen Management System

A full-stack web application for managing canteen operations, including customers, food items, orders, suppliers, stock, and reporting. Built with React, Node.js/Express, and MySQL.

## Features
- User authentication (Admin/Staff)
- Manage customers, food items, orders, suppliers, and stock
- Stock management with triggers
- Reporting (JOINs, aggregates, views)
- Responsive, professional UI/UX with light/dark themes
- Role-based access (Staff cannot access Reports)
- Animated dashboard and micro-interactions

## Tech Stack
- Frontend: React, Material UI
- Backend: Node.js, Express
- Database: MySQL

## Setup
1. Install dependencies in both `backend` and `frontend` folders
2. Set up the MySQL database using `database/schema.sql`
3. Configure environment variables in `backend/.env`
4. Run backend and frontend servers
